

# 🧠 DSA Instructor Bot

A Streamlit-based AI chatbot that answers **only Data Structures & Algorithms (DSA)** questions in a clear, beginner-friendly way.  
This project is designed to help students prepare for coding interviews and strengthen DSA fundamentals.

---

## 🚀 Features

- 🤖 AI-powered DSA instructor
- 📚 Answers **only Data Structures & Algorithms** related questions
- 🧑‍🎓 Beginner-friendly explanations with examples
- 🌐 Web-based UI using Streamlit
- 🔐 Secure API key handling via Streamlit Secrets
- ☁️ Easily deployable on Streamlit Cloud

---

## 🖼️ UI Preview

**DSA Instructor Bot Interface**
- Clean chat-style UI
- User asks a DSA question
- Bot responds with structured explanations
